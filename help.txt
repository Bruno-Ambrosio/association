Iniciar Redis:

    sudo service redis-server start

Iniciar projeto:

    bin/dev ou rails s

Executar script seeds:

    rails db:seed

Baixar dependencias(gemFile):

    bundle install

Iniciar o solargraph:

    solargraph socket

Fazer migrate do banco:

    rails db:migrate

Ligar rails cache

    bin/rails dev:cache
